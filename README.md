# bun-elysia-react-starter

Starter template for projects using a stack of Bun, Elysia and React

## Get Started

1. To initialize your project run `bun run init` (warning, it will overwrite the contents of this file).
1. Install dependencies by running `bun install`.
1. Install Helm charts by running `helm dependency update` in `k8s/db`.
1. Then edit your secrets file with `sops edit`.
1. Finally add a remote PGP key to GitHub and `.sops.yaml`.
